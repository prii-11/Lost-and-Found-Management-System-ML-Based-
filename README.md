# 🔍 Lost and Found Management System (Full Stack + ML)

A smart full-stack web application that helps users report, track, and recover lost items within a campus or organization. This system combines modern web technologies with a Machine Learning–based matching engine to improve accuracy in identifying potential matches.

---

## 🚀 Project Status
- 🔄 Full-stack development
- 🔄 ML-based matching integration
- 🔄 UI/UX improvements in progress

- 🔐 Secure user authentication (Login/Register)

- 🔔 Real-time match notifications
- 📊 Admin dashboard for system control
- 🤖 ML-based intelligent matching system

---

## 🧠 Machine Learning Integration

The system uses a Machine Learning model to enhance matching:

- **Feature Extraction:**  
  Extracts important keywords from item name and description using NLP techniques

- **Similarity Matching:**  
  Uses TF-IDF vectorization and cosine similarity to compare items

- **Smart Matching:**  
  Suggests best matches automatically instead of relying on exact matches

---

## 🧮 Matching Logic

- Text similarity (name + description)
- Category matching
- Location proximity
- Date relevance

➡️ ML improves accuracy over traditional rule-based systems
